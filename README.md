# FuzzyGCN
The PyTorch implementation of ''[Fuzzy Neural Network for Representation Learning on Uncertain Graphs](https://ieeexplore.ieee.org/document/10571582)''.

## Requirements

- python            3.6.5
- numpy             1.19.5
- torch             1.7.1
- torch-geometric   1.7.2
- torch-scatter     2.0.7
- torch-sparse      0.6.9

## Usage
> python train.py --dataset <dataset_name>
